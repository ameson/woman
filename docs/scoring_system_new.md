# 相亲指数评分系统（2024版）

## 评估维度

系统从以下五个维度对用户进行综合评估：

### 1. 形象魅力 (Appearance)
- 穿着打扮风格
- 个人形象维护
- 分数区间：60-100分

### 2. 学历背景 (Education)
- 最高学历
- 专业特长
- 分数区间：60-100分

### 3. 事业成就 (Career)
- 月收入水平
- 职业发展阶段
- 分数区间：60-100分

### 4. 生活品质 (Lifestyle)
- 居住状况
- 生活方式
- 分数区间：60-100分

### 5. 情感态度 (Attitude)
- 婚恋态度
- 家庭角色定位
- 分数区间：60-100分

## 分数计算规则

### 单题得分规则
每个问题提供5个选项，对应分数：
- 最优选项：100分
- 次优选项：90分
- 中等选项：80分
- 及格选项：70分
- 基础选项：60分

### 维度得分计算
每个维度的得分为该维度下所有问题的算术平均值：
```javascript
维度得分 = (问题1得分 + 问题2得分) / 2
```

### 总分计算
1. 基础总分：五个维度得分的算术平均值
```javascript
基础总分 = (形象魅力 + 学历背景 + 事业成就 + 生活品质 + 情感态度) / 5
```

2. 城市调整：
- 一线城市：-10分
- 新一线城市：-5分
- 二线城市：不调整
- 三线城市：+5分

3. 最终总分：
```javascript
最终总分 = Math.min(100, Math.max(0, 基础总分 + 城市调整分))
```

## 注意事项
1. 所有分数都会四舍五入到整数
2. 维度分数和总分都不会超过100分或低于0分
3. 城市调整分数在最后计算，不影响单个维度的得分
4. 评语系统采用鼓励性表达，即使分数较低也保持积极正面的态度
5. 每个维度的评语都包含emoji图标，增加趣味性和可读性
